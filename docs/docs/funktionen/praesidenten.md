# Präsident

Das **Präsidentenspiel** ist ein jährliches Sonderspiel, das den Vereinspräsidenten für das laufende Jahr bestimmt. Der Gewinner erhält eine besondere Auszeichnung in der App.

## Präsidentenspiel erfassen

Das Präsidentenspiel wird wie ein normales Spiel angelegt, jedoch mit dem **Präsidenten-Flag** aktiviert:

1. Navigiere zu **Spiele** am aktiven Abend
2. Lege ein neues Spiel an oder nutze eine entsprechende Spielvorlage
3. Aktiviere die Option **Präsidentenspiel** (🎯) im Spiel-Formular
4. Trage den Gewinner ein und schließe das Spiel ab

:::info
Pro Jahr kann nur ein Spiel als Präsidentenspiel markiert werden. Der Gewinner wird automatisch als Vereinspräsident für das jeweilige Jahr gespeichert.
:::

## Präsidentenliste

Die komplette Präsidentenhistorie ist unter **Verein** → **Präsidenten**-Tab einsehbar:

- Jahresübersicht mit Name des Präsidenten
- Hervorhebung des aktuellen Präsidenten

## Präsidenten-Badge in der Historie

Abende, an denen ein Präsidentenspiel stattgefunden hat, sind in der **Historie** mit einem 🎯-Badge markiert.

## Spielvorlage konfigurieren *(Admin)*

Um das Präsidentenspiel bequem anlegen zu können, kann der Admin unter **Verein** → **Spielvorlagen** eine Vorlage mit gesetztem **Präsidenten-Flag** anlegen. Diese Vorlage erscheint dann beim Anlegen eines Spiels hervorgehoben.
