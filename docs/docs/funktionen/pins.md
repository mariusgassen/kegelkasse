# Vereinsnadeln

**Vereinsnadeln** sind Auszeichnungen des Clubs (z. B. eine goldene Nadel für langjährige Mitgliedschaft). Träger einer Nadel können am Abend per Knopfdruck bestraft werden, falls die Vereinsregeln eine solche Strafe vorsehen.

## Nadeln verwalten *(Admin)*

Navigiere zu **Verein** → **Nadeln**-Tab.

### Nadel erstellen

1. Tippe auf **+ Nadel**
2. Fülle das Formular aus:

| Feld | Beschreibung |
|------|-------------|
| **Name** | Bezeichnung der Nadel (z. B. "Goldene Nadel") |
| **Emoji / Icon** | Visuelles Symbol für die Nadel |
| **Straftyp** | Welche Strafe bei Trägern ausgelöst wird |
| **Träger** | Das Mitglied, das die Nadel aktuell trägt |

3. Tippe auf **Speichern**

### Nadel bearbeiten oder löschen

Tippe auf eine Nadel in der Liste, um sie zu bearbeiten oder zu löschen.

## Nadeln am Abend

### Alert bei anwesenden Trägern

Wenn ein Nadel-Träger als Spieler zum Abend hinzugefügt wird, erscheint ein **Hinweis-Banner** im Abend-Tab. So weiß das Kassenwart sofort, dass eine Nadel-Strafe fällig sein könnte.

### Strafe per Knopfdruck eintrafen

Direkt aus dem Nadel-Alert heraus kann die hinterlegte Strafe für den Träger mit einem einzigen Tipp eingetragen werden — ohne die Strafenmaske öffnen zu müssen.

### Pin-Icons neben Spielernamen

Im Spieler-Tab des Abends erscheinen die Icons aller Pins, die ein Spieler trägt, direkt neben seinem Kegelnamen. Ebenso wird das 🎯-Präsidenten-Badge inline angezeigt.
