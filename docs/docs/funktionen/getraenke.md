# Getränke

Das Getränke-Modul protokolliert Bier- und Schnappsrunden während eines Abends. Die Daten fließen in die Kassenübersicht und Jahresstatistiken ein.

## Runde hinzufügen

1. Im Abend: Tippe auf **Getränke** → **Runde hinzufügen**
2. Wähle die **Art**:
   - 🍺 **Bier**
   - 🥃 **Schnaps**
3. Optional: **Sorte** eingeben (z. B. „Augustiner Hell", „Jägermeister")
4. Wähle die **teilnehmenden Spieler**
5. Tippe auf **Speichern**

## Runde bearbeiten

1. Tippe auf eine Runde in der Liste
2. Ändere Art, Sorte oder Spieler
3. Speichern

## Runde löschen

Tippe auf eine Runde → **Löschen**

Runden werden **weich gelöscht** (Soft-Delete).

## Auswertung

Die Getränkerunden erscheinen in:

- **Kasse**: Anzahl Bier- und Schnappsrunden pro Spieler
- **Jahresstatistiken**: Bier- und Schnappsrunden im Jahresvergleich über alle Abende

## Hinweis zur Abrechnung

Getränke werden aktuell **nicht finanziell** berechnet — es wird nur die Anzahl der Runden protokolliert. Die Abrechnung von Getränkekosten erfolgt manuell außerhalb der App.
