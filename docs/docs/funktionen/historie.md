# Historie

Die **Historie** archiviert alle abgeschlossenen Abende. Sie ermöglicht einen Rückblick auf vergangene Spieltage und bietet Verwaltungsfunktionen für bereits archivierte Abende.

## Historienansicht

Navigiere zu **Historie** (📚) in der Navigation.

Die Liste zeigt alle **geschlossenen Abende** mit:
- Datum
- Spiellokal
- Anzahl der Spieler
- Gesamtsumme der Strafen

## Abend-Detailansicht

Tippe auf einen Abend, um die vollständige Übersicht zu sehen:

- Alle **Spieler** des Abends
- Alle **Spiele** mit Ergebnissen
- Das vollständige **Strafenprotokoll**
- Alle **Getränkerunden**
- Die **Kassenübersicht** mit Ranking

## Abend wiedereröffnen

Falls Korrekturen nötig sind:

1. Tippe auf den Abend in der Historie
2. Tippe auf **Wiedereröffnen**
3. Der Abend ist wieder aktiv und bearbeitbar

Nach den Korrekturen kann er erneut geschlossen werden.

## Nachtragen

Vergessene Einträge können nachgetragen werden ohne den Abend komplett wiederzueröffnen:

1. In der Detailansicht: Tippe auf **Nachtragen**
2. Füge fehlende Strafen, Spiele oder Getränke hinzu
3. Speichern

## Abend löschen

:::danger Achtung
Das Löschen ist **endgültig** und kann nicht rückgängig gemacht werden. Alle Spiele, Strafen und Getränke des Abends werden dauerhaft entfernt.
:::

1. Tippe auf den Abend → **Löschen**
2. Bestätige die Aktion
