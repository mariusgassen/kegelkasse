# Start-Dashboard „Für dich"

Das **Start-Dashboard** (🏠) ist die persönliche Startseite der App. Sie fasst auf einen Blick
zusammen, was für dich heute wichtig ist — ohne dass du erst durch die einzelnen Bereiche
navigieren musst.

Das Dashboard wird beim Öffnen der App automatisch angezeigt, solange **kein Kegelabend aktiv
läuft**. Läuft gerade ein Abend, springt die App direkt in den Abend (die **AKTIV**-Pille im
Kopfbereich zeigt das an). Über den **Start**-Tab (🏠) in der Navigation erreichst du das
Dashboard jederzeit wieder.

## Was zeigt das Dashboard?

| Bereich | Inhalt |
|---------|--------|
| **Begrüßung** | Persönliche Anrede mit deinem Kegelnamen. |
| **Kegelabend läuft** | Erscheint nur, wenn gerade ein Abend aktiv ist — ein Tipp springt zum aktiven Abend. |
| **📅 Nächster Termin** | Der nächste geplante Spielabend mit Datum, Uhrzeit und Lokal. Du kannst direkt **zu- oder absagen** (RSVP), ohne den Termine-Bereich zu öffnen. Über **Alle Termine** gelangst du zur vollständigen Terminliste. |
| **💳 Mein Konto** | Dein aktueller Kontostand (offener Betrag, Guthaben oder „Alles bezahlt"). **Zur Kasse** öffnet dein Konto in der Kasse. |
| **📣 Neuigkeiten** | Die neuesten Ankündigungen und Kegelfahrten. Ein Tipp öffnet den jeweiligen Eintrag im Neuigkeiten-Bereich. |
| **🎳 Meine Saison** | Dein Wurfschnitt der Saison samt kleiner Trendkurve. **Statistik** öffnet die ausführliche Auswertung. |
| **Schnellzugriffe** | Direkte Kacheln zu Termine, Kasse, Neuigkeiten und Statistik. |

## Hinweise

- Das **Mein Konto**-Feld erscheint nur, wenn dein Benutzerkonto mit einem Vereinsmitglied
  verknüpft ist.
- Die **Neuigkeiten**- und **Meine Saison**-Bereiche werden ausgeblendet, wenn es (noch) keine
  Inhalte bzw. keine erfassten Würfe gibt.
- Alle Angaben stammen aus den bereits vorhandenen Bereichen (Termine, Kasse, Neuigkeiten,
  Statistik) — das Dashboard ist eine kompakte Übersicht, keine getrennte Datenquelle.
