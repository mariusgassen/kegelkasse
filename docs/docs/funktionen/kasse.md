# Kasse

Die **Kasse** gibt einen vollständigen Überblick über die Strafen und Getränke eines Abends — mit Ranking, Gesamtsumme und Exportfunktion.

## Kassenübersicht

Navigiere zu **Kasse** (💰) in der Navigation.

Die Übersicht zeigt:

| Spalte | Beschreibung |
|--------|-------------|
| **Rang** | Position nach Gesamtbetrag (höchste Strafe = Platz 1) |
| **Spieler** | Name des Spielers |
| **Strafen (€)** | Gesamtsumme aller Strafen in Euro |
| **Spiele** | Anzahl der Spielsiege |
| **Bier** | Anzahl Bierrunden |
| **Schnaps** | Anzahl Schnappsrunden |

## Gesamtsumme

Am Ende der Übersicht wird die **Gesamtsumme aller Strafen** aller Spieler angezeigt.

## Export

Die Kassenübersicht kann als Text exportiert werden:

1. Tippe auf **Teilen** oder **Kopieren**
2. Der Inhalt wird als formatierter Text exportiert (z. B. für WhatsApp oder Notizen)

Beispiel-Export:
```
Kegelabend 15.03.2025
──────────────────────
1. Max Mustermann   8,50 €
2. Hans Meier       5,00 €
3. Peter Schmitt    3,00 €
──────────────────────
Gesamt: 16,50 €
```

## Mitgliedskonten

Neben der Abendübersicht gibt es **Mitgliedskonten** für die langfristige Abrechnung:

- Admins können Zahlungen (Einzahlungen / Abbuchungen) erfassen
- Der Saldo zeigt an, ob ein Mitglied noch Schulden hat oder Guthaben besitzt
- Einsehbar unter **Verein → Konten** (Admin-Bereich)

:::tip
Die Mitgliedskonten sind ideal für Vereine, die die Strafen am Jahresende oder monatlich abrechnen.
:::
