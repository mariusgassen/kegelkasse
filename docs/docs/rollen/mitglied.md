# Rolle: Mitglied

Als **Mitglied** nimmst du aktiv am Kegelabend teil. Du kannst Spiele erfassen, Strafen vergeben und Getränke protokollieren.

## Navigation

Die App hat fünf Navigationsbereiche, die für Mitglieder alle zugänglich sind:

| Symbol | Bereich | Beschreibung |
|--------|---------|-------------|
| 🎳 | **Abend** | Aktiver Kegelabend |
| 💰 | **Kasse** | Finanzübersicht & Ranking |
| 📚 | **Historie** | Vergangene Abende |
| 📊 | **Statistiken** | Jahresranking |
| ⚙️ | **Verein** | Vereinsinfos (Lesen) |

---

## Profil verwalten

1. Tippe oben rechts auf dein **Profilsymbol**
2. Hier kannst du:
   - **Namen** ändern
   - **Avatar** (Emoji) auswählen
   - **Sprache** wechseln (Deutsch / Englisch)
   - **Passwort** ändern
   - Dich **abmelden**

---

## Am Abend teilnehmen

### Abend auswählen

1. Tippe auf **Abend** in der Navigation
2. Wähle einen offenen Abend aus der Liste

Falls noch kein Abend existiert, kann jedes Mitglied einen neuen anlegen.

### Neuen Abend anlegen

1. Tippe auf **Abend** → **Neuer Abend**
2. Wähle ein **Datum**
3. Optional: **Spiellokal** überschreiben und **Notiz** hinzufügen
4. Tippe auf **Erstellen**

### Spieler hinzufügen

1. Im Abend: Tippe auf **Spieler hinzufügen**
2. Wähle einen Stammspieler aus der Liste **oder** gib einen neuen Namen ein
3. Der Spieler erscheint in der Spielerliste

### Teams bilden

1. Tippe auf **Teams** → **Team hinzufügen**
2. Gib dem Team einen Namen
3. Weise Spieler dem Team zu

---

## Spiele erfassen

1. Tippe auf **Spiele** → **Spiel hinzufügen**
2. Wähle eine **Vorlage** oder erstelle ein freies Spiel
3. Trage den **Gewinner** (Spieler oder Team) ein
4. Optional: **Punktzahlen** je Spieler/Team eintragen
5. Tippe auf **Speichern**

:::tip Eröffnungsspiel
Das Eröffnungsspiel (erkennbar am Kronensymbol 👑) bestimmt den **König** des Abends. Der Gewinner erhält die Königsauszeichnung.
:::

### Spielstatus

Spiele durchlaufen drei Zustände:

```
Offen → Läuft → Beendet
```

- **Offen**: Spiel angelegt, noch nicht gestartet
- **Läuft**: Spiel läuft — Startzeit wird aufgezeichnet
- **Beendet**: Ergebnis eingetragen — Verliererstrafen werden automatisch vergeben

---

## Strafen vergeben

### Einzelstrafe

1. Tippe auf **Strafen** → **Strafe hinzufügen**
2. Wähle den **Strafentyp**
3. Wähle einen oder mehrere **Spieler**
4. Ändere optional den **Betrag**
5. Tippe auf **Speichern**

### Teamstrafe

1. Wähle beim Hinzufügen einer Strafe **Team** statt einzelner Spieler
2. Die Strafe wird automatisch auf alle Teammitglieder aufgeteilt

### Drehtrommel (Zufällige Strafe)

1. Tippe auf das **Drehtrommel-Symbol** 🎡
2. Die Drehtrommel wählt zufällig einen Strafentyp aus
3. Bestätige oder wiederhole

### Strafen bearbeiten / löschen

- Tippe auf eine Strafe in der Liste
- Ändere Betrag oder Anzahl
- Oder tippe auf **Löschen** (Soft-Delete, jederzeit wiederherstellbar)

---

## Getränke protokollieren

1. Tippe auf **Getränke** → **Runde hinzufügen**
2. Wähle die **Art** (Bier / Schnaps)
3. Optional: **Sorte** eingeben (z. B. "Augustiner")
4. Wähle die **teilnehmenden Spieler**
5. Tippe auf **Speichern**

---

## Kasse einsehen

Die **Kasse** zeigt eine Übersicht aller Strafen des aktuellen Abends:

- **Ranking**: Wer hat wie viel zu zahlen?
- **Gesamt**: Gesamtsumme aller Strafen
- **Getränke**: Anzahl Bier- und Schnappsrunden pro Spieler

Der Text kann per **Teilen** oder **Kopieren** exportiert werden.

---

## Eigene Statistiken (Profil)

Im Profil findest du deine **persönlichen Jahresstatistiken**:

- Anzahl besuchter Abende
- Gesamte Strafensumme
- Anzahl Strafenpunkte
- Spielsiege
- Bier- und Schnappsrunden

---

## Abend schließen

1. Im Abend: Tippe auf das **Menü** (drei Punkte)
2. Wähle **Abend schließen**
3. Der Abend wird archiviert und in der **Historie** sichtbar

:::caution
Geschlossene Abende können in der Historie wieder geöffnet werden.
:::
